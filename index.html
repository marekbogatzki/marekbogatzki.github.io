Dlaczego nie używam GitHuba?

Przestałem używać GitHuba do celów prywatnych. Jest to mój wyraz buntu przeciwko centralizacji bazy kodu, a także procesu jego budowania, śledzenia błędów, dokumentacji i dystrybucji. Uważam, że taka centralizacja sprawia, iż awarie GitHuba (podobnie jak AWS czy Cloudflare) mają duży wpływ na ogromną liczbę projektów — nie tylko hobbystycznych. Moje konto na GitHubie wykorzystuję wyłącznie do pracy zawodowej oraz udziału w projektach open source.

Why don't I use GitHub?

I stopped using GitHub for private purposes. This is my way of protesting against the centralization of code repositories, as well as the processes of building, bug tracking, documentation, and distribution. I believe that such centralization means that GitHub outages (similarly to AWS or Cloudflare) have a major impact on a huge number of projects — not only hobby ones. I now use my GitHub account solely for professional work and for participating in open-source projects.
